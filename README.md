  <h2> 🏡 프로젝트 개요 </h2>
  <p> 동떨마켓은 유통기한이 임박하여 어쩔 수 없이 폐기해야 할 상품을 떨이 상품으로 저렴하게 판매할 수 있도록 도와주는 서비스입니다. </p>
  <p>환경 보호뿐만 아니라, 동네 상권 활성화에 이바지하며 물가상승으로 힘들어하는 사람들의 부담을 덜어줍니다. </p>
